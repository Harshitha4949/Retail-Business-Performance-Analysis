# 📊 Retail Business Performance & Profitability Analysis

## 📌 Project Overview

This project analyzes retail business performance using the **Sample Superstore** dataset. The data was cleaned, explored, and analyzed in **Google Colab** using Python, and an interactive dashboard was created in **Power BI** to visualize key business insights.

---

## 🎯 Objectives

- Analyze sales and profit trends.
- Identify high-performing product categories.
- Understand regional sales performance.
- Visualize business KPIs.
- Build an interactive dashboard for decision-making.

## 📊 Key Insights

- Technology category generated the highest sales.
- Office Supplies had the highest number of orders.
- The Western region contributed the highest revenue.
- Discounts significantly affected profitability.
- Some sub-categories produced high sales but low profit.
